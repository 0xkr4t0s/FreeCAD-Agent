# FreeCAD AI Agent Sidebar

Dockable FreeCAD sidebar for sending active document context to local AI coding agents.

<img width = '300' src = '../Media/Header.webp' />


## What

The MVP detects Codex CLI, opens a right-side Qt dock, captures concise FreeCAD state, and streams agent output without blocking the FreeCAD UI.

## Why

The addon is the first slice of a larger SRS for a FreeCAD-aware AI agent workflow. Later slices will add local documentation retrieval, agent skills, richer topology context, and safe macro execution loops.
